== Description ==

This plugin helps you configure your WordPress website to use Marketizator. After setting up an account at Marketizator.com, you simply enter your tracking code in the plugin's configuration page and you're ready to go.

Marketizator is the simplest way to convert your visitors into prospective customers or buyers. It helps increase your conversion or subscription rate.

[Marketizator website](http://www.marketizator.com)

== Installation ==

Sign up at [Marketizator.com](http://www.marketizator.com).

1. Install Marketizator WordPress plugin on your blog
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Enter your Marketizator tracking code in the plugin's settings page. 

All geek stuff are done. You can start using Marketizator!
